# 使用方法ガイド（Jumbory 備品管理）

## 1. できること
- 備品の登録・検索・更新・削除
- 団（グループ）単位での備品管理
- 指導者・スカウト情報の管理
- CSVによる一括登録（スカウト・備品）

## 2. 基本操作（トップページ）
### 備品を登録する
1. 「新しい備品を登録」フォームに入力
2. カテゴリをプルダウンで選択
3. 「登録」をクリック

### 備品を検索する
1. 「検索・フィルタ」の入力欄に備品名またはカテゴリを入力
2. 必要に応じてステータス（保管中/貸出中/要メンテ）を選択

### 備品を編集・削除する
1. 一覧の「編集」をクリック
2. ステータス・数量・カテゴリ・持参有無を更新
3. 「保存」で反映
4. 「削除」で対象備品を削除

## 3. 指導者管理ページ
- トップページ右上の「指導者管理」から遷移
- 指導者の登録、編集、論理削除、復元が可能

## 4. スカウト管理ページ
- トップページ右上の「スカウト管理」から遷移
- スカウトの登録、編集、論理削除、復元が可能
- CSV一括登録に対応

## 5. カテゴリ運用ルール（現行）
- 選択肢はカテゴリマスタ（`categories`）から取得
- 備品登録時はプルダウンから選択
- 不要カテゴリは無効化/削除で運用

## 6. CSV取り込みの注意
- ファイル拡張子は `.csv`
- ヘッダー行を含める
- 不正行がある場合、エラー行をスキップして取り込む実装のため、取り込み結果メッセージを必ず確認

## 7. よくある確認ポイント
- 一覧が出ない場合: API URL と CORS 設定を確認
- カテゴリが出ない場合: `categories` テーブルの `is_active=true` を確認
- 反映されない場合: ブラウザ再読込（キャッシュ）を確認
